# PromptForge v1.7

**Transform weak prompts into battle-ready AI specifications.**

Turn a rough idea like *"write me a blog post about AI"* into a structured RTCF, CO-STAR, or ROSES prompt — all generated by AI via OpenRouter.

👉 **Try it:** [https://lerfin.github.io/PromptForge/](https://lerfin.github.io/PromptForge/)

## Features

### 🎨 Design
- **Split-panel layout** — Input panel (sticky) + output panel (independent scroll) side by side on desktop
- **Playful bouncy animations** — Spring-eased card entrances, button ripples, micro-interactions, scroll reveals
- **Enhanced toast** — Slide-in with progress bar and spring bounce
- **Visual polish** — Gradient buttons, animated focus rings, `::selection` styling, `prefers-reduced-motion` support

### 🚀 Forge
- **Batch Forge 2x** — Generate two variations in parallel, compare side by side, pick the winner
- **Prompt Coach** — "Enhance" button sends your raw prompt to AI for improvement before forging
- **Variable Injection** — Use `{{variable}}` placeholders in your prompt; inline chips appear for filling values
- **Prompt Playbooks** — Save/load/share full forge configurations (goal, tone, format, complexity, custom prompt)
- **A/B Test Mode** — Select two playbooks, run them simultaneously on the same input, compare results

### 🔧 Core
- **AI-powered forge** — Goal/tone/complexity → structured multi-section prompt via OpenRouter
- **3 output formats** — RTCF, CO-STAR, ROSES
- **Re-roll** — Forge again with the same input
- **Prompt chaining** — Use output as input for refinement
- **History** — Auto-saved with filters, star, reorder, delete, export/import
- **Custom templates** — Save forged prompts as reusable templates
- **Context presets** — Save and insert reusable context blocks
- **Keyboard shortcuts** — `Ctrl/Cmd+Enter` to forge, `Ctrl+1-6` to copy sections, `/` to search, `Escape` to reset
- **PWA** — Installable standalone app; service worker for offline fallback
- **Dark theme** — Linear-inspired green-accent dark UI, mobile responsive

## Usage

1. Open the [GitHub Pages link](https://lerfin.github.io/PromptForge/)
2. Enter your OpenRouter API key (stored locally in your browser)
3. Paste a raw prompt, optionally use `{{variable}}` placeholders
4. Choose Goal / Tone / Format / Complexity
5. Click **Forge** (or **Enhance** to improve input first, or toggle **2x** for variations)
6. Copy individual sections or the full prompt

## Local Development

```bash
node pf-server.js
# Serves on http://localhost:8080
```
