# PromptForge

**Transform weak prompts into battle-ready AI specifications.**

Turn a rough idea like *"write me a blog post about AI"* into a structured RTCF, CO-STAR, or ROSES prompt with Role, Task, Context, Format, and more — all generated by AI via OpenRouter.

👉 **Try it:** [https://lerfin.github.io/PromptForge/](https://lerfin.github.io/PromptForge/)

## Features

- **AI-powered forge** — Select a goal (Coding, Writing, Business, etc.), tone, complexity, and model; the app sends your raw prompt to OpenRouter and returns a structured multi-section prompt
- **3 output formats** — RTCF (Role/Task/Context/Format), CO-STAR (Context/Objective/Style/Tone/Audience/Response), ROSES (Role/Objective/Style/Examples/Sensibility)
- **Curated model tiers** — Low-Tier (free/cheap), Mid-Tier (balanced), High-Tier (most powerful). Shared key locked to Gemma 3 12B; personal key unlocks all models
- **Re-roll** — Forge again with the same input
- **Prompt chaining** — Use the forged output as input for another refinement
- **Copy sections** — Copy individual cards or the full prompt as formatted text
- **History** — Auto-saved with search, star, reorder, and delete
- **Export/Import** — Download history as JSON or import from file
- **Custom templates** — Save forged prompts as reusable templates
- **Context presets** — Save and insert reusable context blocks
- **Collapsible cards** — Collapse/expand output sections
- **Font size control** — Adjust output text size
- **Custom system prompt** — Override the default AI prompt via Advanced settings
- **Session restore** — Goal/tone/complexity/model/format persist across reloads
- **Share as URL** — After forging, the URL hash encodes the output — share the link
- **Cost estimator** — Shows estimated token usage and cost per forge
- **Stats** — Total forges, today count, most-used goal
- **Keyboard shortcuts** — `Ctrl/Cmd+Enter` to forge, `Ctrl+1-6` to copy sections, `Ctrl+Shift+C` to copy full prompt, `Escape` to reset, `/` to search history
- **Auto-save draft** — Raw prompt text is saved and restored automatically
- **PWA** — Installable as a standalone app on supported browsers
- **Dark theme** — Linear-inspired green-accent dark UI, responsive for mobile

## Usage

1. Open the [GitHub Pages link](https://lerfin.github.io/PromptForge/)
2. Enter your OpenRouter API key (or use the shared key for limited access)
3. Paste a raw prompt, choose Goal / Tone / Model / Complexity / Format
4. Click **Forge**
5. Copy individual sections or the full prompt

## Local Development

```bash
node pf-server.js
# Serves on http://localhost:8080
```

## Tech

Single-file HTML app — no build step, no dependencies. Uses OpenRouter API for AI generation.
